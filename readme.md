💧 AquaControl

Aplicación Web Administrativa para gestión de negocios de agua purificada.

AquaControl permite administrar pedidos, inventario, gastos y reportes desde cualquier dispositivo usando únicamente navegador web.

---

✨ Características

🏠 Dashboard Inteligente

- Saludo personalizado al administrador
- Fecha actual automática
- Temperatura actual usando API pública
- KPIs en tiempo real
- Últimos movimientos

---

📦 Gestión de Productos

- Registro de productos
- Precio compra / venta
- Stock inicial
- Stock actual automático
- Stock mínimo
- Reabastecimiento inteligente
- Detección automática de productos duplicados

---

🧾 Gestión de Pedidos

- Registro de pedidos
- Asociación con clientes
- Selección automática por inventario disponible
- Cálculo automático:
  - Venta
  - Costo
  - Ganancia
- Descuento automático del inventario
- Exportación PDF

---

💸 Gestión de Gastos

- Registro de gastos
- Categorías
- Agrupación mensual
- Exportación PDF profesional
- Gastos separados por mes

---

👥 Gestión de Clientes

- Nombre
- WhatsApp
- Domicilio
- Edición de datos

---

📊 Reportes

- Reporte mensual PDF
- Reporte de pedidos PDF
- Reporte de gastos PDF
- Logo institucional
- Nombre del administrador
- Diseño profesional

---

- Tema claro
- Tema oscuro

- Mobile First
